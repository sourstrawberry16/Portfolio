Home Down Payment

This is an example of a Visual Basic .NET
application in Visual Studio.
The program allows the user to enter multiple values for a down payment, and then uses
the input to calculate and display how much
money the user would save each year,
and if they have saved enough.

Some skills and aspects in this program are Visual Basic, UI design, user input, arithmetic calculation, and if-else statements.
